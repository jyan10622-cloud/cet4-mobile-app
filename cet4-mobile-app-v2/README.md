# CET4 80/20 备考小程序 · 升级版

这是一个为你当前情况定制的移动端四级备考工具：

- 出社会 4 年
- 成人本科在读
- 有一定词汇和阅读基础
- 输出偏弱，写句子不稳
- 学习时间有限，希望每天用手机碎片化推进
- 希望用 80/20 方法，只学最值钱的内容

## 这次升级了什么

相比最初的原型，这个版本已经补成一个可直接使用的 **轻量 PWA 应用**，核心升级包括：

### 1. 真正可运行的完整目录结构
- `index.html`
- `assets/css/styles.css`
- `assets/js/data.js`
- `assets/js/store.js`
- `assets/js/utils.js`
- `assets/js/app.js`
- `manifest.json`
- `service-worker.js`
- `assets/icons/`

### 2. 每日任务系统升级
- 自动生成当天 4 个高回报任务
- 支持 **完成 / 跳过 / 延期**
- 延期任务会在后续日期以“补做任务”出现
- 支持每日复盘记录

### 3. 词汇模块升级
- 分组：动词 / 形容词 / 短语 / 写作句型
- 状态：**不认识 / 认识 / 已掌握**
- 优先适配你的高频弱项

### 4. 练习模块升级
- 分类自测：词汇 / 语法 / 阅读 / 听力策略 / 写作模板
- 自动记录正确率
- 自动形成错题本
- 句子改错 + 中译英练习，专门补你的输出短板

### 5. 进度显示升级
- 今日完成
- 连续打卡
- 最近 12 周热力图
- 本周学习时长目标
- 鼓励型“预计状态”提示

### 6. 移动端体验升级
- 底部标签栏
- 深色模式
- PWA 安装
- 本地存储
- 支持离线缓存静态资源

## 如何使用

### 本地直接打开
直接双击 `index.html` 或用浏览器打开即可。

### 推荐部署方式
更推荐部署到静态网站后，用手机打开并添加到主屏幕：

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

## 本地数据说明
应用默认使用浏览器本地存储保存：

- 每日任务状态
- 词汇掌握状态
- 自测记录
- 错题本
- 每日复盘
- 设置项

你也可以在“我的”页面里导出 JSON 备份，再导入恢复。

## 当前边界
这个版本仍然是轻量前端版，不包含：

- 真题音频播放
- 云端同步账号
- AI 作文批改
- 微信登录
- 后端题库管理

但已经足够作为一个手机端日常备考工具使用。

## 最适合的下一步
如果你后面还想继续升级，最值得做的是：

1. 迁移成微信小程序

3. 增加作文批改和句子纠错
4. 增加云端同步
