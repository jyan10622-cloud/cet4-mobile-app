# 给 Codex 的下一轮升级提示词

请基于当前目录中的 PWA 版 CET4 备考应用，继续向“更像正式产品”的方向升级。

## 当前用户画像
- 中文用户
- 出社会 4 年
- 成人本科在读
- 有一定英语基础
- 阅读输入还行，但输出偏弱
- 现在最需要补：主谓一致、基础时态、定语从句识别、短句输出
- 目标：用手机每天学习，稳步冲四级

## 当前已有功能
- 首页：倒计时、今日进度、预计状态、低回报提醒
- 今日：自动任务、微课、延期补做、复盘
- 词汇：分组卡片，三档状态
- 练习：分类小测、错题本、改错、中译英
- 计划：12 周路线
- 我的：热力图、统计、设置、导出/导入
- PWA / 本地存储 / 深色模式

## 下一轮优先升级方向

### A. 产品层
1. 增加“周视图”和“月视图”打卡日历
2. 增加“今日推荐任务”权重算法，优先推弱项和错题
3. 增加“复习队列”，让不认识和错题自动回炉
4. 增加“预计完成率”与阶段提醒
5. 增加更强的鼓励反馈，但不要夸张承诺

### B. 学习内容层
1. 把词库扩到 200+ 高频词
2. 增加 30+ 改错题
3. 增加 30+ 中译英短句
4. 增加更贴近四级的短阅读练习
5. 增加作文模板库与常见替换句

### C. 交互层
1. 为词汇卡加“翻卡模式”
2. 为练习模块加“继续上次练习”
3. 为设置页加“清空数据”二次确认
4. 增加过渡动画，但保持轻量
5. 更好地适配 390px 以下手机宽度

### D. 技术层
1. 拆出更清晰的组件化结构
2. 增加基本单元测试或最小 smoke test
3. 优化 service worker 缓存策略
4. 减少首屏 JS 体积
5. 为未来迁移到微信小程序保留数据层接口

## 输出要求
1. 给出升级后的目录结构
2. 给出所有新文件和修改文件
3. 保持中文界面
4. 不要引入重型后端
